# API Dockerizada

## Uso Rápido
```bash
cp .env.example .env
# Edita .env con tus credenciales de Docker Hub
./setup.sh
```

## Configuración GitHub
Secrets necesarios:
- `DOCKER_USERNAME`
- `DOCKER_PASSWORD`

## Configuración Telegram Bot
Secrets necesarios:
- `TELEGRAM_CHAT_ID`
- `TELEGRAM_BOT_TOKEN`

API disponible en: http://localhost:3000
